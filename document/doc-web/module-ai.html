ai

ai命令

1. 基本使用

ai命令的专用语法如下：

ai <command> [options1|options2|options3...]

2. 命令列表

命令执行 含义
ai csv 读取数据文件中的Array数组数据转换成csv的文件格式并输出。
ai data 专用数据生成器，可生成Object/Array两种数据格式。
ai key 为输入数据中的Array或Object追加UUID格式的字段，字段可配置，默认为field = key属性。
ai uk 检查输入数据中是否包含了field1,field2,field3属性中的重复数据，直接输出结果。
ai uuid 随机生成一定数量的UUID字符串，并且拷贝到剪切板中（MacOs）。

3. 数据通用格式

带data节点

{
    "data": [
         {
             "name": "Lang1",
             "email": "lang.yu1@hpe.com"
         },
         {
             "name": "Lang2",
             "email": "lang.yu2@hpe.com"
         }
    ]
}

直接格式

[
     {
         "name": "Lang1",
         "email": "lang.yu1@hpe.com"
     },
     {
         "name": "Lang2",
         "email": "lang.yu2@hpe.com"
     }
]

使用该命令将读取的Array数组数据转换成csv的文件格式

ai csv -p <path>

3. 选项

短参 全参 类型 默认 含义

4.1. 数据文件格式

「通用（略）」数据部分必须对等，从第一个元素开始执行所有的字段解析，如果字段不匹配则直接跳过。
