## 3.参数说明

| 参数格式 | 必填 | 说明 |
| :--- | :--- | :--- |
| -m, --module | 否 | 【FORM】默认模块种类 |
| -p, --path | 否 | 【UI.Form】生成文件名称 |
| -y, --yes | 否 | 【false】是否重写该组件 |

特殊说明：

* 参数m的值主要有：
  * `FORM`：添加表单专用Form；
  * `EDIT`：编辑表单专用Form；
  * `FILTER`：搜索表单专用Form；
  * `HALF`：（保留）
